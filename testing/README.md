# Testing
This is a module for practising in unit testing.

### Requirements
* [java](https://www.java.com/en/) installed
* [maven](https://maven.apache.org/) installed


### Installation
* Clone
```
git clone https://github.com/constantinazouni/labassignment.git
```
* Change path
```
cd ./labassignment
```

* Run
```
mvn test
```
